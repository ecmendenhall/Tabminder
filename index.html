About Tabminder

Tabminder is a simple Chrome extension designed to make you more mindful of browsing distracting sites. It prevents quick visits to Reddit, Twitter, and Hacker News from turning into work-sucking time vortexes by providing a gentle reminder every few minutes. When you open a distracting tab, Tabminder starts counting down from a preset time limit. When time's up, it will prompt you to close the tab or restart the timer. This is just annoying enough to intercept distracted browsing, but not so strict that it prevents you from taking a break once in a while. (You deserve it, right?)

Get Tabminder

Tabminder is available in the Chrome web store. If you're smarter than me and want to tinker with the source (you probably are), you can download Tabminder as an unpackaged extension by forking or cloning the project repository or clicking the icons on the left.

How to use Tabminder

Tabminder is my first Chrome extension, so there will be bugs. If you're lucky enough to find one, please report it and I'll try to fix it.

Once you've added it to Chrome, you'll see a stopwatch icon in the toolbar:

Visit your extension settings and open the options page:

or click on the toolbar icon and select 'Settings':

Add a few distracting sites to the list of timesinks and a time limit in minutes for each.

Make sure you enter each site's hostname as it appears in the address bar.
Note: you might need to double-check your URLs. You don't need to enter 'http://' before the hostname, but you do need to include 'www' if it appears in the Chrome address bar (e.g. entering 'reddit.com' won't work, while entering 'www.reddit.com' will). If the timer doesn't start when you expect it to, double-check your hostnames. I'm working on better filtering in future updates.

When you visit a distracting site, the icon will turn red:

If you check "Show remaining time in toolbar" in the options page, the icon will also display the number of minutes remaining:

While you're browsing, you can add a site to the list of timesinks by clicking the toolbar icon and selecting "Add page to timesinks." Or, if you've found yourself on a distracting site, but don't want to add it permanently, you can start a temporary timer by selecting "Start timer."

Both these options use the default time limit from the settings page:

If you've been browsing a distracting site for too long, Tabminder will redirect you to a reminder page:

If you realize you're wasting time, click "close tab" and get back to work. If you're reading something really interesting, click "restart timer" and the countdown will start over again.
